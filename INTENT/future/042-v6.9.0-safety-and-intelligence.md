## Core Feature: intent-guard (Intent 013)

**Policy enforcement without automation.**

### What It Does

Watches for risky patterns BEFORE execution:
- Destructive commands (`rm -rf`, `dd`, etc)
- System modifications (`pacman -Rns`, etc)
- Network exposure (`chmod 777`, world-readable files)
- Privilege escalation patterns

### How It Works
```bash
$ rm -rf ~/important-stuff
‚ö†Ô∏è  intent-guard: Destructive command detected
   Pattern: rm -rf on home directory
   Risk: HIGH
   Last used: Never
   
   Continue? [y/N]
```

**Philosophy:** Never blocks, only warns. You always have final say.

### Implementation

- Shell integration (preexec hook)
- Pattern matching engine
- Learning from confirmations
- Configurable risk levels
