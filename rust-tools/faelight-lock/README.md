Screen locker for Faelight Forest using swaylock with themed colors.

## Usage
```bash
# Lock screen
faelight-lock

# Health check
faelight-lock --health-check

# Show version
faelight-lock --version
```

## Features

- Pulls colors from faelight-core Theme
- Consistent Faelight Forest aesthetics
- Graceful error handling
- swaylock wrapper with optimized settings

## Dependencies

- swaylock (Wayland screen locker)
- faelight-core theme library

v1.0.0 - Production-ready themed screen locker
